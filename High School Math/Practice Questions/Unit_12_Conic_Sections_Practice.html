Q8HARD难Honors荣誉级 🇨🇦 ON安 ON Provincial-style安大略省考风格 §5 Conic Discriminant圆锥曲线判别式 $B^{2} - 4AC$ · HSG-GPE.A.3 (+) [9 marks][9 分]

Recall the general second-degree equation in two variables $Ax^{2} + Bxy + Cy^{2} + Dx + Ey + F = 0$. The conic discriminant $\Delta_{\text{conic}} = B^{2} - 4AC$ classifies the (non-degenerate) conic: $< 0 \Rightarrow$ ellipse (or circle if $A = C$ and $B = 0$); $= 0 \Rightarrow$ parabola; $> 0 \Rightarrow$ hyperbola. Hyperbola / ellipse / discriminant classification is not in the standard ON MCR3U / MHF4U syllabus; this is honors-extension content.回顾二元一般二次方程 $Ax^{2} + Bxy + Cy^{2} + Dx + Ey + F = 0$。圆锥曲线判别式 $\Delta_{\text{conic}} = B^{2} - 4AC$ 用于辨识（非退化）圆锥曲线类型：$< 0 \Rightarrow$ 椭圆（当 $A = C$ 且 $B = 0$ 时为圆）；$= 0 \Rightarrow$ 抛物线；$> 0 \Rightarrow$ 双曲线。双曲线 / 椭圆 / 判别式分类不在安大略 MCR3U / MHF4U 标准大纲内，属荣誉级拓展内容。

(a) Identify $A$, $B$, $C$ and compute $\Delta_{\text{conic}}$ for $4x^{2} + 9y^{2} - 16x + 18y - 11 = 0$. Classify the conic.对 $4x^{2} + 9y^{2} - 16x + 18y - 11 = 0$，识别 $A$、$B$、$C$ 并计算 $\Delta_{\text{conic}}$，判断曲线类型。 [3]
(b) Identify $A$, $B$, $C$ and compute $\Delta_{\text{conic}}$ for $x^{2} - y^{2} + 6x - 4y + 4 = 0$. Classify the conic.对 $x^{2} - y^{2} + 6x - 4y + 4 = 0$，识别 $A$、$B$、$C$ 并计算 $\Delta_{\text{conic}}$，判断曲线类型。 [2]
(c) Identify $A$, $B$, $C$ and compute $\Delta_{\text{conic}}$ for $y^{2} - 4x - 8y + 20 = 0$. Classify the conic.对 $y^{2} - 4x - 8y + 20 = 0$，识别 $A$、$B$、$C$ 并计算 $\Delta_{\text{conic}}$，判断曲线类型。 [2]
(d) For the equation in part (a), explain in one sentence why the test $A = C$ would (or would not) further reduce the ellipse to a circle.就 (a) 中的方程，用一句话说明 $A = C$ 检验为何能（或不能）进一步把该椭圆退化为圆。 [2]
